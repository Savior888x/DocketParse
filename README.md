# DocketParse
Universal Docket Parser for structuring court data PDF Analyzer &amp; Redaction for extracting and sanitizing records Ledger Engine for tracking settlements, liens, and discrepancies Autofill Module for generating forms from structured data Public Entity Interface aligned with Los Angeles Superior Court   Built to operate with $0 resources
